# Weather App

## Overview
This is a simple weather web app that allows users to search for a city and view the current weather.

## API Used
This app uses the Open-Meteo Geocoding API and Forecast API.

## Features
- Search for a city
- View temperature
- View wind speed
- See time of weather data

## How to Run
1. Download the files
2. Open index.html in a browser OR use Live Server in VS Code
